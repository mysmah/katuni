## Окончания

* -i - существительные / местоимения / числительные

* -o - прилагательные / порядковые числительные

* -a - наречие

* -e - глагол (без указания времени - инфинитив)

* -u - все остальное

* -s - множественное число

* -t - винительный падеж

* -n - притяжательный падеж
